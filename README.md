Simulacion de un cajero automático que realiza reintegros utilizando montículos (Colas de prioridad)

## Archivos De P1

-Act1.cpp Contiene la funcion que nos pide hacer
-Act1.hpp Contiene la clase y operadores necesarios para poder hacer la actividad correctamente
-Caja.dat Contiene los datos de los billetes disponibles

Para poder ejecutar el ejercicio el orden de instrucciones son:
-g++ -std=c++17 -O2 -o cajero Act1.cpp Act1.hpp
- ./cajero

Introducimos la cantidad a reintegrar y nos dará la solución.
